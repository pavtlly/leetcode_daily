**题目链接：**[643. 子数组最大平均数 I](https://leetcode-cn.com/problems/maximum-average-subarray-i/)

**题解：**加加减减 更新答案

```c++
class Solution {
public:
    double findMaxAverage(vector<int>& nums, int k) {
        int n = nums.size(), sum = 0;
        double ans = -1e9;
        for (int i = 0; i < n; i++) {
            if (i < k) sum += nums[i];
            else sum += nums[i] - nums[i - k];
            if (i >= k - 1) ans = max(ans, 1.0 * sum / k);
        }
        return ans;
    }
};
```
